# Word 문서 작성 가이드

## 스타일 시스템
- 제목 1~4 스타일로 문서 구조 체계화
- 본문 스타일: 줄간격 1.15~1.5, 단락 간격 6~12pt
- 스타일 수정: 홈 → 스타일 → 우클릭 → 수정
- 스타일 기반 목차 자동 생성: 참조 → 목차

## 페이지 설정
- 용지: A4 (210 x 297mm)
- 여백: 보통(25.4mm) 또는 좁게(12.7mm)
- 머리글/바닥글: 페이지 번호, 문서 제목, 날짜
- 구역 나누기: 레이아웃 → 나누기 → 다음 페이지

## 표 작성
- 표 스타일: 디자인 탭 → 미리 정의된 스타일
- 열 너비 자동 맞춤: 레이아웃 → 자동 맞춤
- 표 반복 행: 레이아웃 → 반복할 머리글 행
- 수식: 레이아웃 → 수식 (=SUM(ABOVE) 등)

## 참조 기능
- 각주/미주: 참조 → 각주 삽입
- 상호 참조: 그림/표/제목에 대한 참조 링크
- 인용/참고 문헌: 참조 → 인용 삽입
- 캡션: 그림/표 아래 자동 번호 매기기

## 검토 및 협업
- 변경 내용 추적: 검토 → 변경 내용 추적
- 메모: 검토 → 새 메모
- 비교: 검토 → 비교 → 두 문서 비교
- 문서 보호: 검토 → 편집 제한

## 고급 기능
- 메일 병합: 편지, 봉투, 레이블 일괄 생성
- 매크로: 반복 작업 자동화
- 필드 코드: `{DATE}`, `{PAGE}`, `{NUMPAGES}`
- 양식 컨트롤: 개발도구 → 체크박스, 드롭다운
