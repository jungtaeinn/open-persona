# 한글(HWP) 문서 작성 가이드

## 문서 구조
- 스타일: 개요 1~10, 본문, 표제목, 그림제목
- 문단 번호: 1. → 가. → 1) → 가) → (1) → (가)
- 쪽 설정: 편집 용지 A4, 여백 위 20 아래 15 좌 30 우 30
- 머리말/꼬리말: 쪽 번호, 문서 제목

## 표 작성
- 표 만들기: `Ctrl+N,T` 또는 입력 → 표 → 표 만들기
- 셀 병합: 블록 선택 → `M`
- 셀 나누기: 셀 선택 → `S`
- 표 캡션: 표 → 캡션 달기
- 표 자동 채우기: 숫자, 날짜 등 시퀀스

## 글자 모양 (단축키)
- 글꼴: `Alt+L`
- 크기: `Alt+Shift+E`(크게), `Alt+Shift+R`(작게)
- 진하게: `Ctrl+B`
- 기울임: `Ctrl+I`
- 밑줄: `Ctrl+U`
- 글자색: 서식 → 글자 모양 → 글자 색

## 문단 모양
- 줄 간격: 160% (기본), 공문서 180%
- 들여쓰기: 첫줄 10pt
- 문단 간격: 위 0, 아래 0 (기본)
- 정렬: 양쪽, 왼쪽, 오른쪽, 가운데

## 공문서 작성 규정
- 글꼴: 바탕, 10pt
- 줄 간격: 180%
- 문단 번호 체계: 1. → 가. → 1) → 가) → (1) → (가)
- 쪽 번호: 가운데 아래, "- 1 -" 형식
- 수신자: 수신 ○○○장 (경유)
- 발신자: 발신자명 직인

## 유용한 단축키
- 새 문서: `Ctrl+N`
- 저장: `Ctrl+S`, 다른이름: `Ctrl+Shift+S`
- 표 만들기: `Ctrl+N,T`
- 각주: `Ctrl+N,N`
- 찾기: `Ctrl+F`, 바꾸기: `Ctrl+H`
- 글자 수 세기: `Ctrl+Q,I`
- 맞춤법 검사: `F8`
- 인쇄 미리보기: `Ctrl+P` 또는 `Alt+P`
