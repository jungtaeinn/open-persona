# 한글(HWP) 표, 양식, 고급 기능 가이드

## 표 만들기

**기본 표 삽입:**
1. `입력` → `표` → `표 만들기` 또는 단축키 `Ctrl+N,T`
2. 줄(행) 수와 칸(열) 수 지정
3. `만들기` 클릭 → 표 삽입

**표 그리기:**
- `입력` → `표` → `표 그리기` → 마우스 드래그로 자유롭게 표 생성
- 불규칙한 셀 구조가 필요한 경우 유용

**표 이동/크기 조절:**
- 표 선택 (표 바깥 클릭 후 표 테두리 선택) → 드래그로 이동
- 표 모서리 드래그 → 전체 크기 조절
- `Ctrl` + 방향키 → 표 위치 미세 조정

**표 속성:** 표 선택 → `표` → `표/셀 속성` 또는 단축키 `P`

## 셀 합치기/나누기

**셀 합치기:**
1. 합칠 셀 범위를 블록 선택 (마우스 드래그 또는 `F5` + 방향키)
2. `표` → `셀 합치기` 또는 단축키 `M`
3. 선택된 모든 셀이 하나로 합쳐짐

**셀 나누기:**
1. 나눌 셀에 커서 배치
2. `표` → `셀 나누기` 또는 단축키 `S`
3. 나눌 줄 수와 칸 수 지정 → `나누기`

**행/열 추가:**
- 행 추가: 마지막 셀에서 `Tab` → 새 행 자동 추가
- 행/열 삽입: 셀 우클릭 → `줄/칸 추가하기` → 위/아래/왼쪽/오른쪽 선택

**행/열 삭제:**
- 삭제할 행/열 선택 → `표` → `줄/칸 삭제`

## 표 크기 조절

**셀 너비/높이 균등 분배:**
1. 균등 배분할 셀 범위 선택
2. `표` → `셀 너비를 같게` 또는 `셀 높이를 같게`

**개별 셀 크기 조절:**
- 셀 테두리에 마우스 올려 ↔ 커서 표시 → 드래그
- 정확한 수치: `표/셀 속성` (P) → 셀 탭 → 너비/높이 직접 입력

**표 전체 크기 조절:**
- `표/셀 속성` → 표 탭 → 너비/높이 지정
- `글자처럼 취급` 체크: 표가 텍스트처럼 줄 내에 배치
- `표 위치`: 쪽 기준 절대 위치 지정 가능

**자동 크기 조절:**
- `표` → `자동 크기 조절` → `내용에 맞게`, `창에 맞게` 선택

## 셀 속성 설정

**설정 경로:** 셀 선택 → `표` → `표/셀 속성` → `셀` 탭 (단축키 `P`)

**테두리 설정:**
- 각 변(위/아래/좌/우/대각선)별 선 종류, 두께, 색상 지정
- 없음, 실선, 점선, 이중선 등 선택 가능
- 블록 선택 후 일괄 적용 가능

**배경색 설정:**
- `셀` 탭 → `배경` → 색 선택 (단색, 그라데이션, 그림)
- 머리글 행: 진한 배경 + 흰 글씨 (가독성)

**셀 여백:**
- 셀 내부 텍스트와 셀 테두리 사이의 간격 (기본 좌우 1.8mm)

**세로 정렬:**
- 위쪽, 가운데, 아래쪽 정렬

## 자동 채우기

**숫자 자동 채우기:**
1. 연속 숫자가 필요한 셀 범위 블록 선택
2. `표` → `자동 채우기` → `숫자` → 시작값, 증가값 설정
3. `채우기` → 선택 범위에 순차 번호 입력

**지원 자동 채우기 유형:**
- **숫자**: 1, 2, 3... 또는 사용자 지정 시작/증가값
- **날짜**: 2024-01-01, 2024-01-02... (일/월/년 단위)
- **요일**: 월, 화, 수...
- **사용자 지정 목록**: 직접 정의한 시퀀스

## 표 캡션

**캡션 달기:**
1. 표 선택 (표 안 클릭)
2. `표` → `캡션 달기`
3. 위치: 표 위 또는 표 아래
4. 캡션 내용 입력 (예: "[표 1] 연도별 매출 현황")
5. 자동 번호 매기기: `입력` → `캡션 번호` → 번호 자동 증가

**캡션 번호 형식:**
- `표 1`, `[표 1]`, `<표 1>` 등 사용자 지정 가능
- 새 캡션 삽입 시 자동으로 다음 번호 부여
- 중간 삽입/삭제 시 `Ctrl+A` → `F9` (필드 업데이트)로 번호 재정렬

## 블록 계산식

표 안에서 간단한 수식 계산을 수행할 수 있다.

**계산식 입력:**
1. 결과를 표시할 셀에 커서 배치
2. `표` → `블록 계산식` → 함수 선택
3. 계산 범위 지정 → `설정`

**지원 함수:**
| 함수 | 기능 | 예시 |
|------|------|------|
| `SUM` | 합계 | 선택 범위의 합 |
| `AVG` | 평균 | 선택 범위의 평균 |
| `COUNT` | 개수 | 숫자가 있는 셀 수 |
| `MAX` | 최대값 | 선택 범위 최대 |
| `MIN` | 최소값 | 선택 범위 최소 |
| `PRODUCT` | 곱 | 선택 범위의 곱 |

**계산 범위 지정:**
- 블록 선택: 마우스로 셀 범위 드래그
- 왼쪽 셀들, 위쪽 셀들 등 방향 지정

## 셀 주소 표기법

한글 표에서 셀은 엑셀과 유사한 주소 체계를 사용한다.

**주소 규칙:**
- 열: A, B, C, D... (왼쪽부터)
- 행: 1, 2, 3, 4... (위부터)
- 셀 주소: `A1` (1열 1행), `B3` (2열 3행)

**수식에서 셀 주소 사용:**
- 직접 수식 입력: 셀에 `=A1+B1` 형식으로 입력
- 범위: `A1:A10` (A열 1~10행)
- 절대 참조: `$A$1` (복사 시 주소 고정)

**수식 업데이트:**
- 원본 데이터 변경 후 수식 셀 선택 → `F9` 또는 `Ctrl+A` → `F9`

## 차트 삽입 (표 데이터 기반)

**차트 만들기:**
1. 표에서 차트로 만들 데이터 범위를 블록 선택
2. `입력` → `차트` 또는 `표` → `차트 만들기`
3. 차트 종류 선택: 세로 막대, 가로 막대, 꺾은선, 원형, 영역, 분산 등
4. 차트 옵션 설정 → `만들기`

**차트 편집:**
- 차트 더블클릭 → 차트 편집 모드 진입
- 데이터 수정: 데이터 시트 편집
- 차트 종류 변경: 차트 영역 우클릭 → `차트 종류`
- 제목, 범례, 축 레이블 설정

**차트 서식:**
- 색상, 글꼴, 테두리, 그림자 등 개별 요소 서식 지정

## 그리기 마당 (도형, 화살표, 순서도)

**도형 삽입:**
1. `입력` → `그리기 마당` 또는 그리기 도구 모음 사용
2. 도형 종류 선택: 사각형, 원, 삼각형, 화살표, 말풍선, 순서도 도형 등
3. 마우스 드래그로 도형 그리기

**도형 편집:**
- **크기 조절**: 모서리/변 핸들 드래그
- **회전**: 회전 핸들(초록 원) 드래그
- **채우기**: 우클릭 → `개체 속성` → 채우기 색상/패턴/그라데이션
- **선**: 선 색상, 두께, 스타일 변경
- **텍스트 추가**: 도형 더블클릭 → 텍스트 입력

**정렬 및 그룹:**
- 여러 도형 선택 (`Shift+클릭`) → 우클릭 → `맞춤` (수평/수직 정렬, 균등 배치)
- 그룹: 여러 도형 선택 → 우클릭 → `묶기`
- 순서: 앞으로/뒤로 보내기

## 그림 삽입 및 편집

**그림 삽입:**
1. `입력` → `그림` 또는 단축키 `Ctrl+N,I`
2. 이미지 파일 선택 (JPG, PNG, BMP, GIF 등)
3. `넣기` → 문서에 삽입

**그림 배치 방식:**
- `글자처럼 취급`: 텍스트 사이에 문자처럼 삽입
- `자리 차지`: 텍스트가 그림을 감싸며 흐름
- `어울림`: 그림 윤곽을 따라 텍스트 감싸기
- `글 뒤로`: 텍스트 뒤에 배치 (배경 효과)
- `글 앞으로`: 텍스트 위에 배치

**그림 편집:**
- **자르기**: 그림 선택 → `그림` 탭 → `자르기` → 핸들 드래그
- **밝기/대비**: 그림 우클릭 → `개체 속성` → `그림` 탭
- **크기 조절**: `Shift` + 모서리 드래그 (비율 유지)

## 글상자 (텍스트 박스)

**글상자 삽입:**
1. `입력` → `글상자 넣기` 또는 그리기 도구에서 글상자 선택
2. 마우스 드래그로 글상자 영역 지정
3. 글상자 안에 텍스트 입력

**글상자 속성:**
- **테두리**: 선 종류, 두께, 색상 (없음 설정 가능)
- **배경**: 채우기 색상 (없음 = 투명)
- **여백**: 글상자 내부 텍스트 여백
- **글 배치**: 글자처럼 취급, 어울림, 자리 차지 등

**글상자 연결:**
- 여러 글상자를 연결하면 텍스트가 넘칠 때 다음 글상자로 자동 이동
- 신문/잡지 레이아웃에 유용

## 양식 개체

양식 개체는 입력 가능한 대화형 요소로, 디지털 양식 문서를 만들 때 사용한다.

**양식 개체 삽입:** `입력` → `양식 개체` → 유형 선택

**양식 개체 유형:**
- **입력란**: 사용자가 텍스트를 입력할 수 있는 필드
- **체크박스**: 선택/해제 토글 (☑/☐)
- **라디오 버튼**: 여러 옵션 중 하나 선택 (같은 그룹 내 하나만 선택)
- **콤보 상자**: 드롭다운 목록에서 선택
- **명령 버튼**: 클릭 시 매크로 실행

**양식 개체 설정:**
1. 양식 개체 삽입 후 더블클릭 → 속성 대화상자
2. 이름, 기본값, 크기, 제한 조건 등 설정
3. 콤보 상자: 목록 항목 추가/삭제
4. 라디오 버튼: 그룹 이름 지정 (같은 그룹끼리 배타 선택)

**양식 모드:** `보기` → `양식 모드` → 양식 개체만 입력 가능 (본문 편집 잠금)

## 메일 머지 (대량 문서 생성)

메일 머지는 하나의 양식 문서에 데이터를 연결하여 여러 문서를 자동 생성하는 기능이다.

**메일 머지 단계:**
1. **양식 문서 작성**: 고정 텍스트 + 표시 달기(필드) 삽입
2. **표시 달기**: `입력` → `메일 머지 표시 달기` → 필드 이름 설정 (이름, 주소, 직위 등)
3. **데이터 연결**: `도구` → `메일 머지` → 데이터 파일 선택
   - 지원 형식: 한글 표, 엑셀(.xlsx), CSV, ODBC 데이터
4. **필드 매핑**: 표시 달기와 데이터 열 연결
5. **실행**: `메일 머지 만들기` → 프린터 출력 또는 새 문서로 생성

**활용 예:**
- 개별 수신자 이름이 들어간 공문 대량 발송
- 수료증/상장 자동 생성
- 주소 라벨 일괄 인쇄
- 안내문/초대장 개인화 발송

## 매크로 기초

매크로는 반복적인 작업을 자동화하기 위해 일련의 명령을 기록/실행하는 기능이다.

**매크로 기록:**
1. `도구` → `매크로` → `매크로 기록 시작` 또는 단축키 `Alt+Shift+H`
2. 매크로 이름 입력
3. 자동화할 작업 수행 (서식 변경, 텍스트 입력 등)
4. `도구` → `매크로` → `매크로 기록 중지`

**매크로 실행:**
- `도구` → `매크로` → `매크로 실행` → 매크로 선택 → `실행`
- 단축키 지정: 매크로 관리에서 단축키 할당 가능

**스크립트 매크로:**
- 한글 매크로는 자바스크립트 기반 스크립트 지원
- `도구` → `매크로` → `매크로 편집` → 스크립트 직접 작성
- `HAction`, `HParameterSet` 등 한글 전용 API 사용

**주의사항:**
- 매크로 포함 문서: `.hwp` 저장 시 매크로 포함 여부 확인
- 보안: 신뢰할 수 없는 매크로 실행 주의

## 맞춤법 검사 및 교정 부호

**맞춤법 검사:**
1. `도구` → `맞춤법 검사` 또는 단축키 `F8`
2. 오류 발견 시: 수정 제안 목록 표시
3. `바꾸기`, `건너뛰기`, `모두 바꾸기` 선택
4. `사용자 사전`에 추가: 전문 용어 등 오탐 방지

**자동 맞춤법 검사:**
- `도구` → `환경 설정` → `맞춤법` → `자동 맞춤법 검사` 체크
- 오류 단어 아래 빨간 물결선 표시

**교정 부호:**
- `입력` → `교정 부호` → 교정 부호 삽입
- 삽입 부호(∧), 삭제 부호(⌇), 교체, 이동 표시 등
- 문서 검토/교열 과정에서 수정 의견 표시에 사용

**글자 수 세기:**
- `도구` → `글자 수 세기` 또는 단축키 `Ctrl+Q,I`
- 글자, 낱말, 줄, 문단, 쪽 수 표시
- 블록 선택 시 선택 영역만 세기
