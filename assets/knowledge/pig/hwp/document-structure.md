# 한글(HWP/HWPX) 문서 구조 및 스타일 가이드

## 한글 문서 기본 구조

한글 문서는 다음 계층으로 구성된다:

- **용지 (편집 용지)**: 문서의 물리적 크기와 방향
- **여백**: 본문 영역과 용지 가장자리 사이의 간격
- **머리말/꼬리말**: 모든 쪽 상단/하단에 반복 표시되는 영역
- **본문**: 실제 내용이 입력되는 영역
- **각주/미주**: 참조 설명 영역
- **바탕쪽**: 쪽 배경 (워터마크, 로고 등)

**문서 파일 형식:**
- `.hwp`: 한글 97 이후 바이너리 형식
- `.hwpx`: 한글 2014 이후 XML 기반 개방형 형식 (표준)
- `.hwt`: 한글 문서 서식 (템플릿)

## 편집 용지 설정

**설정 경로:** `쪽` → `편집 용지` 또는 단축키 `F7`

**주요 용지 규격:**
| 용지 | 크기 (mm) | 용도 |
|------|-----------|------|
| A4 | 210 × 297 | 일반 문서, 보고서 |
| B5 | 176 × 250 | 소책자, 교재 |
| A3 | 297 × 420 | 포스터, 대형 표 |
| Letter | 216 × 279 | 미국 표준 |
| Legal | 216 × 356 | 법률 문서 |

**설정 항목:**
1. **용지 종류**: 규격 선택 또는 사용자 정의 (너비/높이 직접 입력)
2. **용지 방향**: 세로 (기본) / 가로
3. **제본**: 한 쪽, 양 쪽(맞쪽), 위로 넘기기

## 여백 설정

**설정 경로:** `쪽` → `편집 용지` → `여백` 탭 (F7)

**여백 항목:**
- **위 여백**: 용지 상단 ~ 본문 상단 (기본 20mm)
- **아래 여백**: 본문 하단 ~ 용지 하단 (기본 15mm)
- **좌 여백**: 용지 왼쪽 ~ 본문 왼쪽 (기본 30mm)
- **우 여백**: 용지 오른쪽 ~ 본문 오른쪽 (기본 30mm)
- **머리말**: 용지 상단 ~ 머리말 영역 (기본 10mm)
- **꼬리말**: 꼬리말 영역 ~ 용지 하단 (기본 10mm)
- **제본 여백**: 제본을 위한 추가 여백 (기본 0mm)

**공문서 표준 여백:**
- 위 20mm, 아래 15mm, 좌 20mm, 우 15mm, 머리말 15mm, 꼬리말 15mm

## 다단 편집

**설정 경로:** `쪽` → `다단 설정` 또는 단축키 `Alt+Shift+Enter` (단 나누기)

**다단 설정 방법:**
1. `쪽` → `다단 설정` 클릭
2. 단 수 지정 (2단, 3단 등)
3. 단 너비: 같게 또는 각 단 별도 지정
4. 단 사이 간격: 기본 8mm
5. 단 사이 선: 없음, 실선, 점선 등
6. 적용 범위: 현재 구역 또는 문서 전체

**단 나누기:**
- 현재 위치에서 다음 단으로 강제 이동: `Ctrl+Shift+Enter` 또는 `입력` → `단 나누기`
- 특정 부분만 다단: 해당 영역을 별도 구역으로 설정 후 다단 적용

## 스타일 시스템

한글의 스타일은 글자 모양 + 문단 모양을 하나의 세트로 정의하여 일괄 적용하는 기능이다.

**기본 제공 스타일:**
- `개요 1` ~ `개요 10`: 제목/소제목 계층
- `본문`: 일반 텍스트
- `표제목`, `그림제목`: 캡션용
- `머리말`, `꼬리말`: 해당 영역 전용
- `각주`, `미주`: 참조 전용

**스타일 적용:** 서식 도구 상자에서 스타일 이름 선택 또는 단축키 `Alt+1` ~ `Alt+0` (개요 1~10)

**스타일 편집:**
1. `서식` → `스타일` (단축키 `F6`)
2. 수정할 스타일 선택 → `스타일 편집`
3. 글자 모양, 문단 모양 설정
4. `확인` → 해당 스타일이 적용된 모든 문단에 일괄 반영

**사용자 스타일 추가:**
- `서식` → `스타일` → `스타일 추가` → 이름, 글자/문단 모양 설정 → `추가`

## 글자 모양 설정

**설정 경로:** `서식` → `글자 모양` 또는 단축키 `Alt+L`

**주요 설정 항목:**
- **글꼴**: 한글, 영문, 한자, 일어 등 언어별 개별 지정 가능
- **크기**: pt 단위 (10pt 기본, 공문서 기본)
- **장평**: 글자 너비 비율 (100% 기본, 90%이면 좁게)
- **자간**: 글자 사이 간격 (0% 기본, -5%이면 좁게)
- **글자 색**: 직접 지정 또는 팔레트 선택
- **글자 속성**: 진하게(B), 기울임(I), 밑줄(U), 취소선, 위첨자, 아래첨자
- **그림자/외곽선**: 텍스트 효과
- **커닝**: 글자 쌍 간격 자동 조절

**자주 쓰는 단축키:**
- 진하게: `Ctrl+B`, 기울임: `Ctrl+I`, 밑줄: `Ctrl+U`
- 크기 1pt 증가: `Alt+Shift+E`, 감소: `Alt+Shift+R`
- 글자 모양 대화상자: `Alt+L`

## 문단 모양 설정

**설정 경로:** `서식` → `문단 모양` 또는 단축키 `Alt+T`

**주요 설정 항목:**
- **정렬**: 양쪽 정렬 (기본), 왼쪽, 오른쪽, 가운데, 배분
- **줄 간격**: 글줄 사이 간격
  - 160% (한글 기본), 180% (공문서), 130% (빡빡한 레이아웃)
  - 단위: %, pt, 배수 선택 가능
- **문단 간격**: 문단 위/아래 추가 간격 (기본 0pt)
- **들여쓰기**: 첫 줄 들여쓰기 (기본 10pt), 왼쪽/오른쪽 들여쓰기
- **내어쓰기**: 첫 줄 제외하고 들여쓰기 (번호 목록에 유용)

**줄 간격 단축키:**
- 줄 간격 160%: `Alt+Shift+Z`
- 줄 간격 200%: 직접 설정 필요

## 문단 번호와 글머리 기호

**문단 번호 설정:**
1. `서식` → `문단 번호` 또는 도구 모음에서 번호 아이콘 클릭
2. 번호 모양 선택 (1, 2, 3 / 가, 나, 다 / I, II, III 등)
3. 수준(레벨) 조절: `Tab` (하위), `Shift+Tab` (상위)

**공문서 번호 체계:**
- 1수준: `1.`, `2.`, `3.` ...
- 2수준: `가.`, `나.`, `다.` ...
- 3수준: `1)`, `2)`, `3)` ...
- 4수준: `가)`, `나)`, `다)` ...
- 5수준: `(1)`, `(2)`, `(3)` ...
- 6수준: `(가)`, `(나)`, `(다)` ...

**글머리 기호:** `서식` → `글머리 기호` → 기호 모양 선택 (●, ■, ◆, ▶ 등)

## 탭 설정

**설정 경로:** `서식` → `문단 모양` → `탭 설정` 탭

**탭 유형:**
- **왼쪽 탭**: 텍스트가 탭 위치 오른쪽으로 정렬 (기본)
- **오른쪽 탭**: 텍스트가 탭 위치 왼쪽으로 정렬 (금액 등)
- **가운데 탭**: 텍스트가 탭 위치 중앙 정렬
- **소수점 탭**: 소수점 기준으로 정렬 (숫자 데이터)

**탭 설정 방법:**
1. `서식` → `문단 모양` → `탭 설정` 탭
2. 탭 위치(mm) 입력 → 종류 선택 → `추가`
3. 채움선: 없음, 점선, 실선, 하이픈 (목차 등에 활용)

**단축키:** `Tab` 키로 다음 탭 위치로 이동

## 머리말/꼬리말

**머리말 설정:**
1. `쪽` → `머리말` 또는 단축키로 머리말 영역 진입
2. 텍스트 입력 (문서 제목, 장 제목 등)
3. 서식 지정 (글꼴, 크기, 정렬)
4. `Shift+Esc`로 본문 복귀

**꼬리말 설정:**
1. `쪽` → `꼬리말`
2. 주로 페이지 번호 삽입: `입력` → `쪽 번호 넣기`
3. 형식: `1`, `- 1 -`, `1/전체`, `I, II, III` 등

**구역별 다른 머리말/꼬리말:**
- 구역을 나눈 후 각 구역에서 별도의 머리말/꼬리말 설정 가능
- `양 쪽` 설정 시 홀수 쪽/짝수 쪽 다른 머리말/꼬리말 지원

## 각주/미주

**각주 삽입:**
1. 참조할 위치에 커서 배치
2. `입력` → `각주` 또는 단축키 `Ctrl+N,N`
3. 쪽 하단에 각주 영역 생성 → 내용 입력
4. 본문의 각주 번호 자동 증가 (1, 2, 3...)

**미주 삽입:**
1. `입력` → `미주` 또는 단축키 `Ctrl+N,E`
2. 문서 끝(또는 구역 끝)에 미주 영역 생성

**각주/미주 설정:**
- 번호 모양: 1, 2, 3 / i, ii, iii / *, †, ‡ 등
- 번호 매기기: 연속 또는 쪽마다 새로 시작
- 구분선: 각주와 본문 사이의 선 스타일

## 책갈피 및 하이퍼링크

**책갈피 설정:**
1. 참조 대상 위치에 커서 배치 (또는 텍스트 블록 선택)
2. `입력` → `책갈피` → 이름 입력 → `추가`
3. 문서 내 특정 위치를 이름으로 저장

**하이퍼링크 삽입:**
1. 텍스트 선택 → `입력` → `하이퍼링크` 또는 `Ctrl+K`
2. 연결 대상 선택:
   - **파일**: 외부 문서 연결
   - **URL**: 웹 주소 연결
   - **책갈피**: 문서 내 책갈피 위치로 이동
   - **전자 메일**: 이메일 주소 연결

**상호 참조 삽입:**
- `입력` → `상호 참조` → 참조 대상 (표, 그림, 수식 캡션 등) 선택

## 바탕쪽 (페이지 배경)

바탕쪽은 본문 뒤에 깔리는 배경 레이어로, 워터마크나 고정 로고 등에 사용한다.

**바탕쪽 설정:**
1. `쪽` → `바탕쪽` → 바탕쪽 만들기
2. 바탕쪽 편집 모드 진입 → 그림, 글상자, 도형 삽입
3. 편집 완료 후 `Shift+Esc`로 본문 복귀

**활용 예:**
- 회사 로고 워터마크 (투명도 조절)
- 레터헤드 양식 (상단 로고 + 하단 연락처)
- 페이지 테두리 장식
- "대외비", "초안" 등 워터마크 텍스트

**바탕쪽 vs 머리말/꼬리말:**
- 바탕쪽: 본문 뒤 레이어, 그림/도형 자유 배치 가능
- 머리말/꼬리말: 본문 위/아래 고정 영역, 주로 텍스트/쪽번호

## 구역 나누기 (섹션별 다른 설정)

구역(섹션)을 나누면 같은 문서 내에서 서로 다른 편집 용지, 여백, 다단, 머리말/꼬리말을 적용할 수 있다.

**구역 나누기:**
1. 새 구역을 시작할 위치에 커서 배치
2. `쪽` → `구역 나누기` 또는 `쪽` → `편집 용지`에서 `적용 범위: 새 구역으로`

**구역별 독립 설정 항목:**
- 편집 용지 크기/방향 (예: 특정 구역만 가로 방향)
- 여백
- 다단 설정
- 머리말/꼬리말 내용
- 쪽 번호 시작 번호
- 바탕쪽

**활용 예:**
- 보고서 표지 (여백 없음) + 본문 (표준 여백)
- 세로 본문 + 가로 방향 대형 표 페이지
- 챕터별 다른 머리말 제목
- 부록: 쪽 번호 별도 시작 (A-1, A-2...)
