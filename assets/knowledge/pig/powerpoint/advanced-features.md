# PowerPoint 고급 기능 가이드

## Morph 전환 (변환 전환 효과)

Morph(변환)는 두 슬라이드 사이에서 개체의 이동, 크기, 회전, 색상 변화를 부드럽게 애니메이션하는 전환 효과다.

**지원 변환:**
- **위치 이동**: 개체가 A 지점에서 B 지점으로 슬라이드
- **크기 변화**: 작은 이미지 → 큰 이미지로 확대/축소
- **회전**: 0° → 90° 등 각도 변환
- **색상/투명도**: 색상, 투명도 점진적 변환
- **텍스트 변환**: 단어 단위 또는 문자 단위 변환

**요구 사항:** Office 365 또는 PowerPoint 2019 이상

## Morph 사용법

**기본 단계:**
1. 첫 번째 슬라이드에 개체 배치
2. 슬라이드 복제: 왼쪽 패널에서 우클릭 → `슬라이드 복제`
3. 복제된 슬라이드에서 개체의 위치, 크기, 회전, 색상 등 변경
4. 복제된 슬라이드 선택 → `전환` 탭 → `변환(Morph)` 선택

**개체 매칭 규칙:**
- PowerPoint가 같은 개체를 자동 인식 (같은 유형 + 비슷한 위치)
- 수동 매칭: 두 슬라이드의 개체 이름을 동일하게 설정 (예: `!!shape1`)
- 이름 앞에 `!!`를 붙이면 강제 매칭 (선택 창: `홈` → `선택` → `선택 창`)

**Morph 옵션:**
- `전환` 탭 → `효과 옵션` → `개체`, `단어`, `문자` 중 택

## Zoom (요약, 섹션, 슬라이드 확대/축소)

**Summary Zoom (요약 확대/축소):**
1. `삽입` 탭 → `확대/축소` → `요약 확대/축소`
2. 포함할 섹션/슬라이드 선택 → `삽입`
3. 자동으로 목차 슬라이드 생성 → 클릭하면 해당 섹션으로 이동 후 복귀

**Section Zoom (구역 확대/축소):**
1. `삽입` → `확대/축소` → `구역 확대/축소`
2. 이동할 섹션 선택 → 해당 섹션 썸네일이 슬라이드에 삽입
3. 프레젠테이션 중 클릭 → 섹션으로 이동 → 끝나면 원래 슬라이드로 복귀

**Slide Zoom (슬라이드 확대/축소):**
1. `삽입` → `확대/축소` → `슬라이드 확대/축소`
2. 특정 슬라이드 선택 → 썸네일 삽입
3. Zoom 옵션: 우클릭 → `확대/축소 옵션` → 전환 효과, 복귀 여부 설정

## 섹션 (슬라이드 그룹화)

대규모 프레젠테이션을 논리적 단위로 그룹화하여 관리.

**섹션 추가:**
1. 왼쪽 슬라이드 패널에서 섹션 시작 슬라이드 우클릭
2. `섹션 추가` 클릭
3. 섹션 이름 입력 (예: "1부: 현황 분석")

**섹션 관리:**
- **이동**: 섹션 헤더 드래그 → 섹션 단위로 슬라이드 순서 변경
- **축소/확장**: 섹션 헤더 클릭 → 토글
- **삭제**: 섹션 우클릭 → `섹션 제거` (슬라이드 유지) 또는 `섹션 및 슬라이드 제거`
- **모두 축소**: 아무 섹션 우클릭 → `모두 축소`

## 디자이너 (AI 디자인 아이디어)

**사용 조건:** Microsoft 365 구독, 인터넷 연결.

**사용 방법:**
1. 슬라이드에 텍스트 또는 이미지 삽입
2. 우측에 `디자인 아이디어` 패널 자동 표시 (또는 `디자인` 탭 → `디자인 아이디어`)
3. 제안된 레이아웃 중 선택 → 클릭으로 적용

**디자이너가 활성화되는 경우:**
- 텍스트만 입력한 슬라이드 (자동 레이아웃 제안)
- 이미지 삽입 시 (이미지 배치 + 텍스트 레이아웃 제안)
- 프로세스/목록 텍스트 → SmartArt 변환 제안
- 아이콘/타임라인 등 자동 시각화

**끄기:** `파일` → `옵션` → `일반` → `PowerPoint 디자이너` 체크 해제

## 녹화 기능

**슬라이드 쇼 녹화:**
1. `슬라이드 쇼` 탭 → `슬라이드 쇼 녹화` → `처음부터` 또는 `현재 슬라이드부터`
2. 녹화 화면: 슬라이드 + 카메라 미리보기 + 녹화 컨트롤
3. `녹화` 버튼 클릭 → 나레이션 녹음 시작
4. 슬라이드 넘기며 설명 → 각 슬라이드별 타이밍 자동 저장
5. `중지` 버튼으로 녹화 종료

**녹화 중 사용 가능 도구:**
- **펜/형광펜**: 슬라이드 위에 필기
- **레이저 포인터**: 마우스 포인터를 레이저로 표시
- **카메라**: 발표자 얼굴 녹화 (우하단 원형)
- **지우기**: 잉크 지우기

**녹화 삭제:** 슬라이드 우클릭 → `녹화 지우기` → 타이밍/나레이션 선택

## 내보내기 (동영상, PDF, 이미지, GIF)

**동영상 MP4 내보내기:**
1. `파일` → `내보내기` → `비디오 만들기`
2. 해상도 선택: Ultra HD (4K), Full HD (1080p), HD (720p), Standard (480p)
3. `녹화된 타이밍 및 설명 사용` 또는 `각 슬라이드에 걸린 시간(초)` 설정
4. `비디오 만들기` → 저장 위치/파일명 지정

**PDF 내보내기:**
- `파일` → `내보내기` → `PDF/XPS 문서 만들기`
- 옵션: 슬라이드 범위, 유인물 레이아웃, 메모 포함 여부

**이미지 내보내기:**
- `파일` → `다른 이름으로 저장` → 형식: PNG 또는 JPEG
- `모든 슬라이드` 또는 `현재 슬라이드만` 선택

**GIF 내보내기 (Microsoft 365):**
- `파일` → `내보내기` → `애니메이션 GIF 만들기`
- 품질/크기 선택 → 각 슬라이드 표시 시간 설정

## SmartArt 고급 활용

**주요 SmartArt 유형:**
- **프로세스**: 순서가 있는 단계 표현 (기본 프로세스, 반복 블록 화살표)
- **계층 구조**: 조직도, 트리 구조
- **관계**: 벤다이어그램, 방사형, 기어
- **매트릭스**: 2x2, 4분면
- **피라미드**: 상하 계층, 역피라미드

**SmartArt 삽입 및 편집:**
1. `삽입` → `SmartArt` → 유형 선택
2. 텍스트 창에 내용 입력 (Tab으로 하위 수준)
3. `SmartArt 디자인` 탭 → 레이아웃 변경, 색 변경, SmartArt 스타일
4. 개별 도형 서식 변경: 도형 우클릭 → `도형 서식`

**텍스트 → SmartArt 변환:**
- 글머리 기호 텍스트 선택 → `홈` → `SmartArt로 변환`

## 3D 모델 및 SVG 아이콘

**3D 모델 삽입:**
1. `삽입` → `3D 모델` → `온라인 원본에서` 또는 `파일에서`
2. 3D 모델 선택 → 삽입
3. 마우스 드래그로 회전 (중앙 3D 컨트롤)
4. `3D 모델` 탭 → 미리 정의된 보기(정면, 위, 측면 등) 선택
5. Morph 전환과 결합하면 슬라이드 간 3D 회전 애니메이션 가능

**SVG 아이콘 삽입:**
1. `삽입` → `아이콘` → 카테고리에서 선택
2. 아이콘 크기 조절, 색상 변경: `그래픽 서식` 탭 → `그래픽 채우기`

**SVG 분해 (개별 편집):**
1. SVG 아이콘 선택
2. `그래픽 서식` 탭 → `도형으로 변환`
3. 그룹 해제 (Ctrl+Shift+G) → 개별 도형으로 분리
4. 각 부분별 색상, 크기 독립적으로 변경 가능

## 수식 삽입

**수식 삽입 방법:**
1. `삽입` 탭 → `수식` (π 기호)
2. 미리 정의된 수식 선택 (이차방정식, 이항정리 등) 또는 `새 수식 삽입`
3. `수식` 탭에서 기호/구조 삽입:
   - **분수**: 분모/분자
   - **위/아래 첨자**: 지수, 아래첨자
   - **근호**: 제곱근, n제곱근
   - **적분**: 정적분, 부정적분
   - **행렬**: m×n 행렬

**LaTeX 입력:**
- 수식 입력 상자에서 `{LaTeX}` → `변환` 클릭 (Office 365)
- 예: `\frac{a}{b}`, `\sqrt{x}`, `\sum_{i=1}^{n}`

## 접근성 검사

**접근성 검사 실행:**
1. `검토` 탭 → `접근성 검사`
2. 오류, 경고, 팁 목록 확인 → 각 항목 클릭하면 해결 방법 표시

**주요 점검 항목:**
- **대체 텍스트**: 모든 이미지, 도형, 차트에 대체 텍스트 추가 (우클릭 → `대체 텍스트 편집`)
- **읽기 순서**: `홈` → `정렬` → `선택 창` → 아래에서 위 순서가 읽기 순서
- **표 머리글**: 표의 첫 행을 머리글로 지정
- **슬라이드 제목**: 모든 슬라이드에 제목 개체 틀 포함
- **색상 대비**: 텍스트와 배경 간 충분한 대비 (4.5:1 이상)
- **링크 텍스트**: "여기를 클릭" 대신 설명적 텍스트 사용

## 공동 작업 (실시간 편집)

**OneDrive/SharePoint에 저장:**
1. `파일` → `저장` 또는 `공유` → OneDrive/SharePoint 위치 선택
2. 클라우드에 저장된 파일만 실시간 공동 작업 가능

**공유 및 실시간 편집:**
1. `공유` 버튼 (우상단) 클릭
2. 이메일 주소 입력 → 권한 설정 (`편집 가능` 또는 `보기 가능`)
3. `보내기` 클릭 → 초대된 사용자가 동시에 편집 가능
4. 다른 사용자의 커서/편집 위치가 실시간 색상으로 표시

**댓글 및 @멘션:**
- 텍스트/개체 선택 → 우클릭 → `새 메모` (또는 `Ctrl+Alt+M`)
- 댓글에 `@이름` 입력 → 해당 사용자에게 알림 전송
- `해결` 버튼으로 댓글 스레드 종료

**버전 기록:**
- `파일` → `정보` → `버전 기록` → 이전 버전 열기/복원
