# PowerPoint 마스터 슬라이드 및 템플릿 가이드

## 슬라이드 마스터란?

슬라이드 마스터는 PPT 디자인의 최상위 계층으로, 모든 슬라이드의 공통 서식을 제어한다.

**계층 구조:**
- **슬라이드 마스터** → 전체 프레젠테이션의 공통 디자인 (배경, 로고, 글꼴)
- **슬라이드 레이아웃** → 마스터 아래 개별 배치 유형 (제목, 콘텐츠, 비교, 빈 화면 등)
- **슬라이드** → 실제 내용이 입력되는 개별 슬라이드

**핵심 원리:**
- 마스터를 수정하면 해당 마스터를 사용하는 모든 레이아웃과 슬라이드에 반영
- 레이아웃을 수정하면 해당 레이아웃을 사용하는 슬라이드에만 반영
- 개별 슬라이드에서 직접 수정한 서식은 마스터 변경에 영향받지 않음 (오버라이드)

## 슬라이드 마스터 편집 방법

**마스터 보기 진입:**
1. `보기` 탭 → `슬라이드 마스터` 클릭
2. 왼쪽 패널에 마스터(큰 썸네일)와 레이아웃(작은 썸네일) 표시
3. 최상단의 큰 슬라이드가 마스터, 아래가 레이아웃

**마스터에서 수정 가능한 요소:**
- 배경 색상/이미지
- 로고, 회사명 등 공통 텍스트
- 글꼴 테마, 색상 테마
- 개체 틀(Placeholder) 위치 및 크기
- 페이지 번호, 날짜, 바닥글 위치

**마스터 보기 종료:**
- `슬라이드 마스터` 탭 → `마스터 보기 닫기`

## 슬라이드 레이아웃 추가/수정/삭제

**레이아웃 추가:**
1. 슬라이드 마스터 보기 진입
2. `슬라이드 마스터` 탭 → `레이아웃 삽입`
3. 필요한 개체 틀(제목, 내용, 그림 등)을 배치
4. 레이아웃 이름 변경: 우클릭 → `레이아웃 이름 바꾸기`

**레이아웃 수정:**
1. 원하는 레이아웃 선택
2. 개체 틀 이동, 크기 조절, 추가/삭제
3. 배경이나 도형 추가 가능

**레이아웃 삭제:**
- 사용하지 않는 레이아웃 우클릭 → `레이아웃 삭제`
- 현재 슬라이드에서 사용 중인 레이아웃은 삭제 불가 (먼저 다른 레이아웃으로 변경 필요)

## 공통 요소 설정 (로고, 페이지 번호, 날짜, 배경)

**로고 삽입:**
1. 슬라이드 마스터 보기 → 최상단 마스터 슬라이드 선택
2. `삽입` → `그림` → 로고 이미지 삽입
3. 원하는 위치(우하단 등)에 배치 → 모든 슬라이드에 자동 적용

**페이지 번호 / 날짜 / 바닥글:**
1. `삽입` 탭 → `머리글/바닥글`
2. `슬라이드` 탭에서 설정:
   - `날짜 및 시간`: 자동 업데이트 또는 고정
   - `슬라이드 번호`: 체크
   - `바닥글`: 텍스트 입력
   - `제목 슬라이드에는 표시 안 함`: 필요시 체크
3. `모두 적용` 클릭

**배경 설정:**
- 마스터에서 우클릭 → `배경 서식` → 단색, 그라데이션, 이미지 선택

## 글꼴 테마 설정

**글꼴 테마란?** 제목 글꼴과 본문 글꼴을 하나의 세트로 정의하여 일괄 적용하는 기능.

**설정 방법:**
1. 슬라이드 마스터 보기 진입
2. `슬라이드 마스터` 탭 → `글꼴` 드롭다운
3. 기존 테마 선택 또는 `글꼴 사용자 지정` 클릭
4. **제목 글꼴** (예: Pretendard Bold) 지정
5. **본문 글꼴** (예: Pretendard Regular) 지정
6. 테마 이름 입력 → `저장`

**추천 조합:**
- 제목: 프리텐다드 SemiBold / 본문: 프리텐다드 Regular
- 제목: Noto Sans KR Bold / 본문: Noto Sans KR Regular
- 제목: 나눔스퀘어 ExtraBold / 본문: 나눔스퀘어 Regular

## 색상 테마 설정

**색상 테마 구성 요소:**
- **주조색 2개**: 어두운 배경용 + 밝은 배경용
- **보조색 2개**: 텍스트/배경 보조
- **강조색 6개**: 차트, 도형, 하이퍼링크 등에 사용

**설정 방법:**
1. 슬라이드 마스터 보기 → `슬라이드 마스터` 탭 → `색` 드롭다운
2. `색 사용자 지정` 클릭
3. 12개 색상 슬롯 각각 설정
4. 테마 이름 입력 → `저장`

**색상 선택 팁:**
- 강조색 1: 가장 많이 사용할 브랜드 메인 색상
- 강조색 2: 보조 강조 색상
- 강조색 3~6: 차트에서 데이터 계열 구분용
- 하이퍼링크 색상도 브랜드에 맞게 변경

## 개체 틀 (Placeholder) 관리

**개체 틀 유형:**
- `제목`: 슬라이드 제목 텍스트 영역
- `내용`: 텍스트, 표, 차트, SmartArt, 이미지 중 택 1 삽입
- `텍스트`: 순수 텍스트 전용
- `그림`: 이미지 전용
- `차트`: 차트 전용
- `표`: 표 전용
- `미디어`: 동영상/오디오

**개체 틀 추가/관리:**
1. 슬라이드 마스터 보기 → 레이아웃 선택
2. `슬라이드 마스터` 탭 → `개체 틀 삽입` → 유형 선택
3. 드래그하여 위치/크기 지정
4. 개체 틀 내 기본 텍스트(안내 문구) 편집 가능
5. 서식(글꼴, 크기, 정렬) 설정 → 해당 레이아웃 사용 시 자동 적용

## 테마 저장 및 재사용 (.thmx)

**테마에 포함되는 요소:** 색상 테마, 글꼴 테마, 효과 테마, 배경 스타일.

**테마 저장:**
1. `디자인` 탭 → `테마` 그룹 → 아래 화살표 → `현재 테마 저장`
2. 파일명 입력, 확장자 `.thmx`
3. 기본 저장 위치: `C:\Users\[사용자]\AppData\Roaming\Microsoft\Templates\Document Themes`

**테마 적용:**
- `디자인` 탭 → `테마` → `테마 찾아보기` → .thmx 파일 선택
- Word, Excel에서도 동일한 .thmx 테마 사용 가능 (Office 통일 디자인)

## 템플릿 저장 (.potx)

**템플릿에 포함되는 요소:** 테마 + 슬라이드 마스터/레이아웃 + 기본 슬라이드 내용.

**템플릿 저장:**
1. `파일` → `다른 이름으로 저장`
2. 파일 형식: `PowerPoint 템플릿 (*.potx)` 선택
3. 기본 저장 위치: `C:\Users\[사용자]\Documents\사용자 지정 Office 서식 파일`

**템플릿으로 새 PPT 만들기:**
1. `파일` → `새로 만들기` → `개인` 탭
2. 저장한 템플릿 선택 → 템플릿 기반 새 프레젠테이션 생성

**팁:**
- .potx는 열면 항상 새 파일로 시작 (원본 보호)
- .pptx는 원본 파일 자체를 편집
- 반복 사용할 디자인은 반드시 .potx로 저장

## 여러 마스터 사용

하나의 PPT에서 여러 슬라이드 마스터를 동시에 사용할 수 있다.

**활용 사례:**
- 표지: 마스터 A (어두운 배경, 큰 제목)
- 본문: 마스터 B (밝은 배경, 콘텐츠 중심)
- 마무리: 마스터 C (감사 인사, CTA)

**마스터 추가:**
1. 슬라이드 마스터 보기 진입
2. `슬라이드 마스터` 탭 → `슬라이드 마스터 삽입`
3. 새 마스터에 별도 디자인 적용
4. 마스터 보기 닫기 → 슬라이드별 레이아웃 선택 시 마스터 구분 가능

**다른 PPT에서 마스터 가져오기:**
1. `슬라이드 마스터` 탭 → `테마` → `테마 찾아보기`
2. .thmx 또는 .pptx 파일 선택 → 해당 파일의 마스터가 추가됨

## 마스터 변경이 기존 슬라이드에 미치는 영향

**자동 반영되는 경우:**
- 마스터의 배경색/이미지 변경 → 해당 마스터 사용 슬라이드 전체 반영
- 마스터의 글꼴 테마 변경 → 개체 틀 내 텍스트에 반영
- 마스터의 색상 테마 변경 → 테마 색상 사용 요소에 반영
- 마스터에 로고 추가 → 모든 슬라이드에 로고 표시

**반영되지 않는 경우 (오버라이드):**
- 슬라이드에서 직접 글꼴/크기를 수동 변경한 텍스트
- 슬라이드에서 직접 변경한 배경
- 개체 틀 밖에 삽입한 텍스트 상자/도형

**오버라이드 초기화:**
- 슬라이드 우클릭 → `레이아웃 다시 설정` → 마스터/레이아웃 서식으로 복원
- 개별 텍스트: 선택 → `홈` → `모든 서식 지우기`
