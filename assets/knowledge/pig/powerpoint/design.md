# PowerPoint 디자인 및 레이아웃 가이드

## 슬라이드 레이아웃 원칙
- 하나의 슬라이드에 하나의 메시지 (One Slide, One Message)
- 6x6 규칙: 한 슬라이드에 최대 6줄, 한 줄에 최대 6단어
- 여백(마진) 충분히 확보: 최소 상하좌우 1.5cm
- 시선 흐름: 좌상단 → 우하단 (Z 패턴 또는 F 패턴)

## 텍스트 가이드
- 제목: 24~36pt, Bold
- 본문: 18~24pt, Regular
- 캡션/주석: 12~14pt, Light
- 글꼴 조합: 제목(산세리프) + 본문(세리프) 또는 통일
- 한글 추천: 프리텐다드, 노토산스, 스포카한산스
- 줄간격: 1.2~1.5배

## 색상 가이드
- 주조색 1개 + 보조색 2개 + 강조색 1개
- 배경색과 텍스트 색 대비 명확히
- 일관된 색상 팔레트 유지
- 차트 색상: 구분이 명확한 색조 사용

## 도형 및 아이콘
- 도형 정렬: `맞춤` 도구 활용 (수평/수직 균등배치)
- 그룹화: 관련 요소 묶어서 관리
- SmartArt: 프로세스, 계층, 관계도 등 자동 레이아웃
- 아이콘: 삽입 → 아이콘 (Microsoft 내장) 또는 SVG

## 애니메이션 베스트 프랙티스
- 최소한으로 사용 (내용 전달에 방해되지 않게)
- 나타내기: 페이드, 닦아내기
- 화면 전환: 밀어내기, 변환(Morph)
- 트리거 활용: 클릭 시 특정 요소만 동작

## 차트/표 삽입
- 엑셀 데이터 연결: 원본 데이터 자동 업데이트
- 차트 스타일: PPT 내 디자인 탭에서 일괄 변경
- 표 간소화: 테두리 최소화, 행 교차 색상으로 가독성
