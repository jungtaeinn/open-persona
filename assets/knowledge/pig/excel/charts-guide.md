# Excel 차트 완전 가이드

## 차트 유형별 사용 용도
- 세로 막대형(Column): 항목 간 값 비교 (예: 부서별 매출)
- 가로 막대형(Bar): 항목명이 긴 경우의 비교 (예: 제품명별 판매량)
- 꺾은선형(Line): 시간에 따른 추세 (예: 월별 매출 변화)
- 영역형(Area): 꺾은선 아래 면적 표시, 누적 비교에 적합 (예: 누적 매출 구성)
- 원형(Pie): 전체 대비 비율 (항목 5~7개 이하 권장), 3D 원형은 왜곡 주의
- 도넛형(Doughnut): 원형의 변형, 여러 데이터 계열 비교 가능
- 분산형(Scatter/XY): 두 변수 간 상관관계 분석 (예: 광고비 vs 매출)
- 거품형(Bubble): 분산형에 3번째 변수를 거품 크기로 표현
- 콤보(Combo): 두 가지 차트 유형 결합 (예: 막대+꺾은선)
- 폭포(Waterfall): 증감의 누적 효과 (예: 수익 구조 분석)
- 트리맵(Treemap): 계층적 데이터의 비율 (예: 카테고리별 매출 구성)
- 선버스트(Sunburst): 계층적 비율을 동심원으로 표현 (예: 지역>도시>매장)
- 히스토그램: 데이터 분포 (빈도수), 통계 분석용
- 상자 수염(Box & Whisker): 데이터 분포의 중앙값, 사분위수, 이상치 시각화

## 차트 삽입 및 기본 설정
- 데이터 선택 → 삽입 탭 → 원하는 차트 유형 클릭
- 추천 차트: 삽입 → 추천 차트 — 데이터에 적합한 차트 자동 제안
- 단축키: Alt+F1 (현재 시트에 기본 차트 즉시 삽입), F11 (새 시트에 차트 삽입)
- 차트 이동: 차트 우클릭 → 차트 이동 → 새 시트 또는 다른 시트로 이동
- 차트 크기: 모서리/변 드래그, Alt 누른 채 드래그하면 셀에 맞춤 정렬
- 기본 차트 유형 변경: 차트 우클릭 → 차트 종류 변경 → 원하는 유형 → "기본 차트로 설정"

## 데이터 계열 추가/제거
- 차트 선택 → 차트 디자인 탭 → 데이터 선택 → 추가/편집/제거
- 드래그로 추가: 새 데이터를 선택하여 차트 위에 붙여넣기
- 계열 순서 변경: 데이터 원본 선택 대화상자에서 위/아래 화살표
- 숨겨진 데이터 표시: 데이터 선택 → 숨겨진 셀 및 빈 셀 → 숨겨진 데이터 표시 체크
- 빈 셀 처리: 공백으로 표시 / 0으로 표시 / 선으로 연결 중 선택
- 행/열 전환: 차트 디자인 → 행/열 전환 (데이터 축 방향 변경)

## 축 서식 설정
- 축 더블클릭 또는 우클릭 → 축 서식
- 최소값/최대값: 자동(Auto) 또는 고정값 입력 — 차트 범위 직접 지정
- 주 단위/보조 단위: 눈금선 간격 설정 (예: 주 단위 1000, 보조 단위 500)
- 로그 스케일: 축 옵션 → 로그 눈금 → 밑 지정 (기본 10) — 값 범위 차이가 큰 데이터에 적합
- 축 반전: "항목을 거꾸로" 체크 — 축 방향 뒤집기
- 표시 단위: 없음/백/천/만/백만/억 — 큰 숫자를 간결하게 표시
- 숫자 형식: 축 레이블에 천단위 구분, 백분율 등 적용
- 날짜 축: 축 유형을 "날짜 축"으로 변경 → 기본 단위(일/월/년) 설정, 데이터 없는 날짜도 간격 유지

## 차트 제목, 범례, 데이터 레이블
- 차트 제목: 차트 요소 추가(+) → 차트 제목 → 위치 선택 (위, 겹쳐서)
- 수식 연결: 차트 제목 클릭 → 수식 입력줄에 `=시트명!셀주소` 입력 → 셀 값이 제목으로 동적 반영
- 범례: 차트 요소 추가 → 범례 → 위치 (위/아래/왼쪽/오른쪽)
- 범례 항목 이름 변경: 데이터 선택 → 계열 편집 → 계열 이름 변경
- 데이터 레이블: 차트 요소 추가 → 데이터 레이블 → 위치 (중심, 바깥쪽 끝, 안쪽 등)
- 레이블 내용: 값, 계열 이름, 범주 이름, 백분율 중 선택
- 레이블 서식: 레이블 우클릭 → 서식 → 숫자 형식, 글꼴, 배경색 등

## 추세선 추가
- 데이터 계열 우클릭 → 추세선 추가
- 선형(Linear): `y = ax + b` — 일정한 증감 패턴
- 지수(Exponential): `y = ae^(bx)` — 점점 빠르게 증가/감소
- 로그(Logarithmic): `y = a·ln(x) + b` — 초기 급변 후 완만
- 다항식(Polynomial): 차수 지정 (2~6) — 곡선 형태, 차수 높을수록 데이터에 밀착
- 거듭제곱(Power): `y = ax^b` — 일정 비율로 증가
- 이동평균(Moving Average): 기간 지정 → 단기 변동 제거, 추세 파악
- 수식 표시: 추세선 옵션 → "차트에 수식 표시" 체크
- R-제곱 값: "차트에 R-제곱 값 표시" 체크 — 1에 가까울수록 추세선 적합도 높음
- 예측: "앞으로" 또는 "뒤로" 기간 수 입력 → 미래/과거 값 예측선 연장

## 보조 축 사용
- 용도: 단위나 범위가 다른 두 데이터 계열을 하나의 차트에 표시 (예: 매출액 vs 성장률%)
- 설정: 데이터 계열 우클릭 → 데이터 계열 서식 → 보조 축 선택
- 보조 축이 차트 오른쪽에 표시되며 독립적으로 스케일 조정 가능
- 주의: 두 축의 0 기준선이 다르면 시각적 왜곡 발생 가능 → 축 최소값/최대값 수동 조정
- 범례에 단위 명시 권장 (예: "매출(백만원)", "성장률(%)")

## 콤보 차트
- 삽입 → 차트 → 콤보 또는 차트 종류 변경 → 콤보
- 사용자 지정 조합: 각 데이터 계열별로 차트 유형(막대/꺾은선/영역)과 보조 축 여부 개별 설정
- 대표 예시: 막대(매출액, 기본 축) + 꺾은선(이익률, 보조 축) — 절대값과 비율 동시 표현
- 묶은 막대 + 꺾은선: 항목별 비교와 추세를 한 차트에
- 누적 막대 + 꺾은선: 구성 비율과 합계 추세 동시 표현

## 폭포 차트 (Waterfall)
- 용도: 초기값에서 증가/감소를 거쳐 최종값에 도달하는 과정 시각화
- 삽입: 데이터 선택 → 삽입 → 폭포 차트
- 활용 예: 기초 수익 → (+매출) → (-원가) → (-경비) → 순이익
- 합계 설정: 합계 막대를 더블클릭 → 데이터 요소 서식 → "합계로 설정" 체크
- 색상: 증가(파란색), 감소(빨간색), 합계(회색)가 기본 — 차트 스타일에서 변경 가능
- 음수 값: 자동으로 아래 방향 표시

## 트리맵, 선버스트 차트
- 트리맵: 계층 데이터를 중첩된 직사각형으로 표현, 면적이 값의 크기
- 선버스트: 계층 데이터를 동심원(고리)으로 표현, 안쪽이 상위 레벨
- 데이터 구조: 여러 열로 계층 표현 (예: 대분류 | 중분류 | 소분류 | 값)
- 삽입: 데이터 선택 → 삽입 → 계층 구조 차트 → 트리맵 또는 선버스트
- 활용: 제품 카테고리별 매출 구성, 조직별 인력 분포, 비용 항목 분석

## 조건부 서식과 차트 연동
- 직접 연동은 불가하지만 보조 열을 활용하여 유사 효과 구현
- 방법 1: 기준값 초과/미달을 별도 계열로 분리 → 각 계열에 다른 색상 적용
- 예: B열(기준 초과 값), C열(기준 미달 값) → 두 계열을 다른 색으로 누적 막대 표시
- 방법 2: VBA로 데이터 포인트별 색상 동적 변경 (각 막대 개별 서식)
- 방법 3: 데이터 레이블에 조건부 아이콘 추가 (레이블 내용에 IF+CHAR 수식 활용)

## 스파크라인
- 셀 안에 표시되는 미니 차트 (데이터 옆에 추세를 시각적으로 표현)
- 삽입: 삽입 탭 → 스파크라인 → 꺾은선/세로막대/승패 중 선택
- 데이터 범위: 한 행 또는 한 열의 데이터 (예: B2:M2의 월별 매출)
- 위치 범위: 스파크라인이 표시될 셀
- 고점/저점 표시: 스파크라인 도구 → 디자인 → 고점/저점/첫째점/마지막점/음수점 체크
- 축 옵션: 스파크라인 도구 → 축 → 모든 스파크라인에 동일 최소/최대값 적용 (비교 용도)
- 그룹화: 여러 스파크라인을 그룹으로 묶어 동일 축/서식 적용 → 공정한 비교
- 스파크라인은 인쇄 시에도 표시됨

## 차트 디자인 팁
- 색상: 3~5색 이내 사용, 강조 데이터만 진한 색, 나머지는 회색 톤 (데이터 잉크 비율 최대화)
- 배경: 차트 영역 배경은 흰색 또는 밝은 단색, 눈금선은 연한 회색
- 글꼴: 맑은 고딕/나눔고딕 계열, 크기 10~12pt (제목 14~16pt)
- 3D 차트 지양: 3D 효과는 데이터 왜곡, 2D가 정확한 비교에 적합
- 원형 차트: 항목 5개 이하 권장, 가장 큰 조각을 12시 방향에서 시작
- 축 레이블: 단위를 축 제목에 명시 (예: "매출 (백만원)"), 기울어진 레이블 피하기
- 범례: 가능하면 데이터 옆에 직접 레이블 배치 (범례 왕복 참조 최소화)
- 간결함: 불필요한 테두리, 눈금선, 장식 제거 → 데이터에 집중
- 일관성: 같은 보고서 내 차트는 동일한 색상 팔레트, 글꼴, 스타일 유지
