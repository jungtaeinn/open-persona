# Excel 서식 및 데이터 정리 가이드

## 셀 서식
- 숫자 형식: `#,##0` (천단위 구분), `#,##0.00` (소수점), `0%` (백분율)
- 날짜 형식: `YYYY-MM-DD`, `YYYY년 M월 D일`
- 사용자 지정 형식: `[조건]형식` (예: `[RED]-#,##0;[BLUE]#,##0`)

## 조건부 서식
- 셀 값 규칙: 보다 큰/작은, 사이, 같은
- 데이터 막대: 셀 내 시각적 비교
- 색조: 그라데이션 색상으로 값 분포
- 아이콘 집합: 화살표, 신호등, 깃발 등
- 수식 사용: `=AND(B2>100, C2="완료")` 같은 복합 조건

## 데이터 유효성 검사
- 목록 드롭다운: 셀에 선택지 제한
- 정수/소수 범위: 입력값 범위 제한
- 날짜/시간 범위: 유효 기간 제한
- 사용자 지정 수식: `=LEN(A1)<=20` 같은 검증

## 피벗 테이블
- 행/열/값/필터 영역에 필드 배치
- 값 필드 설정: 합계, 개수, 평균, 최대, 최소
- 그룹화: 날짜(월/분기/연), 숫자 범위
- 계산 필드: 기존 필드 조합으로 새 측정값
- 슬라이서: 시각적 필터링 도구

## 차트 가이드
- 비교: 막대/세로막대 차트
- 추세: 꺾은선 차트
- 비율: 원형/도넛 차트
- 분포: 히스토그램/상자 수염
- 관계: 분산형 차트
- 콤보 차트: 막대 + 꺾은선 조합

## 데이터 정리 패턴
- 텍스트 나누기: 데이터 → 텍스트 나누기 (구분 기호)
- 중복 제거: 데이터 → 중복 제거
- 빈 셀 채우기: `Ctrl+G` → 빈 셀 선택 → 위 셀 참조 수식
- 공백 제거: `TRIM()`, `CLEAN()`
- 일괄 변환: `찾기/바꾸기` + 와일드카드 (`*`, `?`)
