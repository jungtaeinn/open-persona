# Excel 핵심 함수 가이드

## 조건/논리 함수
- `IF(조건, 참값, 거짓값)` — 조건 분기의 기본
- `IFS(조건1, 값1, 조건2, 값2, ...)` — 다중 조건 (Excel 2019+)
- `SWITCH(표현식, 값1, 결과1, ...)` — 값 매칭 분기
- `AND(조건1, 조건2, ...)`, `OR()`, `NOT()` — 논리 조합

## 조회/참조 함수
- `VLOOKUP(검색값, 범위, 열번호, [일치유형])` — 세로 조회 (기본)
- `HLOOKUP(검색값, 범위, 행번호, [일치유형])` — 가로 조회
- `INDEX(배열, 행번호, [열번호])` + `MATCH(검색값, 범위, [일치유형])` — 강력한 2차원 조회 조합
- `XLOOKUP(검색값, 검색범위, 반환범위, [없을때], [일치모드], [검색모드])` — VLOOKUP의 진화 (Microsoft 365)
- `OFFSET(기준셀, 행이동, 열이동, [높이], [너비])` — 동적 범위 참조

## 텍스트 함수
- `LEFT(텍스트, 문자수)`, `RIGHT()`, `MID(텍스트, 시작위치, 문자수)`
- `LEN(텍스트)` — 문자열 길이
- `FIND(찾을문자, 텍스트)` vs `SEARCH()` — 대소문자 구분 여부 차이
- `SUBSTITUTE(텍스트, 이전, 새로운)` — 문자열 치환
- `TEXTJOIN(구분자, 빈셀무시, 범위)` — 텍스트 병합
- `CONCAT(텍스트1, 텍스트2)` — 단순 연결

## 날짜/시간 함수
- `TODAY()`, `NOW()` — 현재 날짜/시간
- `DATE(년, 월, 일)`, `YEAR()`, `MONTH()`, `DAY()`
- `DATEDIF(시작일, 종료일, "Y"/"M"/"D")` — 날짜 차이
- `NETWORKDAYS(시작일, 종료일, [휴일])` — 영업일 계산
- `EOMONTH(시작일, 월수)` — 월말 날짜

## 통계/수학 함수
- `SUMIF(범위, 조건, [합계범위])`, `SUMIFS()` — 조건부 합계
- `COUNTIF()`, `COUNTIFS()` — 조건부 개수
- `AVERAGEIF()`, `AVERAGEIFS()` — 조건부 평균
- `SUMPRODUCT(배열1, 배열2)` — 배열 곱의 합 (고급 집계)
- `RANK.EQ(값, 범위, [순서])` — 순위

## 배열 함수 (동적 배열 / Microsoft 365)
- `FILTER(배열, 조건, [없을때])` — 조건 필터링
- `SORT(배열, [정렬기준], [순서])` — 정렬
- `UNIQUE(배열, [열기준], [한번만])` — 중복 제거
- `SEQUENCE(행수, [열수], [시작], [단계])` — 숫자 시퀀스
- `LET(이름1, 값1, ..., 계산식)` — 변수 정의로 수식 가독성 향상
- `LAMBDA(매개변수, 수식)` — 사용자 정의 함수 (Microsoft 365)
