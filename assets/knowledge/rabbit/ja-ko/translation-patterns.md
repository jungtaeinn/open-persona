# 일본어 → 한국어 번역 패턴 가이드

## 어순 유사성 활용
- 일본어와 한국어는 어순(SOV)이 동일
- 조사 대응: は→은/는, が→이/가, を→을/를, に→에, で→에서
- 경어 체계 유사: です/ます→~입니다/~습니다, だ/である→~이다

## 한자어 활용
- 일본어 한자어 → 한국어 한자어 직접 대응하는 경우 많음
- 確認(かくにん) → 확인, 管理(かんり) → 관리, 会議(かいぎ) → 회의
- 단, 의미가 다른 한자어 주의: 勉強 → 공부(O), 무리하다(X)

## 조사 매핑
- は (wa) → 은/는 (주제)
- が (ga) → 이/가 (주격)
- を (wo) → 을/를 (목적격)
- に (ni) → 에/에게 (방향/대상)
- で (de) → 에서 (장소/수단)
- の (no) → 의 (소유격)
- と (to) → 와/과 (병렬)
- から (kara) → 에서/부터 (출발점)
- まで (made) → 까지 (도달점)

## 경어 변환
- ~です → ~입니다/~이에요
- ~ます → ~합니다/~해요
- ~ございます → ~있습니다 (높임)
- ~いただく → ~받다 (겸양)
- ~くださる → ~주시다 (존경)
- お/ご~ → 접두사 생략 또는 "귀하의"

## 문화적 뉘앙스
- よろしくお願いします → "잘 부탁드립니다" / "잘 부탁해요"
- お疲れ様です → "수고하셨습니다" / "안녕하세요" (인사)
- 失礼します → "실례합니다"
- すみません → "죄송합니다" / "저기요" (상황에 따라)

## IT 용어
- 開発(かいはつ) → 개발
- 実装(じっそう) → 구현
- テスト → 테스트
- デプロイ → 배포 / 디플로이
- コードレビュー → 코드 리뷰
- プルリクエスト → 풀 리퀘스트 / PR
